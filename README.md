# Symbolica

Repositório acadêmico aberto voltado para matemática computacional,
contendo notas de estudo, exercícios resolvidos, experimentos computacionais
e explorações matemáticas.

------------------------------------------------------------
OBJETIVOS
------------------------------------------------------------

Este repositório tem como objetivo reunir materiais de estudo
e desenvolvimento acadêmico relacionados a áreas como:

- Matemática
- Computação
- Estatística
- Machine Learning
- Álgebra Linear
- Análise Numérica
- Algoritmos

A ideia é manter um ambiente aberto para aprendizado,
organização de conhecimento e possível colaboração futura.

------------------------------------------------------------
TECNOLOGIAS UTILIZADAS
------------------------------------------------------------

- LaTeX
- Python
- Jupyter Notebook
- NumPy
- SciPy
- Git

------------------------------------------------------------
FILOSOFIA DO PROJETO
------------------------------------------------------------

O conteúdo deste repositório não possui compromisso
com formalidade editorial.

Muitas notas podem conter:

- ideias em desenvolvimento
- anotações incompletas
- experimentos
- demonstrações alternativas
- comentários pessoais
- observações intuitivas

O objetivo principal é documentar processos de aprendizagem
e exploração acadêmica.
